meta-explain

OBJETIVO

El objetivo es el de construir una meta-explicabilidad de distintos modelos.

Se tendrn distintas tareas y, para cada una de ellas, un modelo entrenado a partir de sus propios datos. El objetivo que se persigue es el de encontrar una estructura comn que sirva como meta-expicabilidad de los modelos. Sera la explicabilidad pero aplicada a (learning1998a?)multitask learning.

Como comentarios:

  • Las variables \(X_1\) y \(X_2\) se obtienen a partir de una distribucin normal multivariante.
  • \(X_3\) se obtiene a partir de una distribucin uniforme entre 0 y 1.
  • \(X_4\) se obtiene a partir de una distribucin normal de media 0 y desviacin tpica 1.
  • \(X_5\) se obtiene a como una transformacin cuadrtica creciente a partir de \(X_4\) aadiendo un error aleatorio normal.
  • La variable objetivo \(Y\) se obtiene con el resultado de aplicar la funcin de Rastrigin para \(X_1\) y \(X_2\) (estandariazado) ms una transformacin cuadrtica utilizando las variables \(X_4\) y \(X_5\).
  • La variable \(X_3\) no interviene en la generacin de la variable objetivo.

En general, el comportamiento de las variables con respecto a la variable objetivo

  • Dos variables estarn muy correladas y tendrn un patrn similar para todas las tareas,
  • Una variable que no contribuya a la variable objetivo y
  • Dos variables que sigan un patrn distinto para cada tarea y correladas.

Idea general:

  • Entrenar un modelo para cada tarea. (TODO: Ahora mismo se entrena cada tarea independiente del resto. Podra ser interesante aplicar multi-task learning)
  • Buscar una estructura general que sea aplicable (en cierto grado) para todas las tareas. Esto debe incluir:
    • Diferenciar variables que siguen un patrn similar para todas las tareas (actualmente seran las variables \(X_1\) y \(X_2\)),
